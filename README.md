# bash-крипт развертывания wg-easy 
(https://github.com/wg-easy/wg-easy)


Возможности:
---
1. Автоопределение ip адреса хоста и заполнение в env WG_HOST контейнера.
1. Задание пароля для WEB UI в скртытом поле ввода , который будет автоматически преобразован в хеш и назанчен в env PASSWORD_HASH контейнера.
2. Указание порта WG (порт самого WEB UI будет на единицу больше)
3. Задание ip диапазона для клиентов
4. По завершении работы скрипта вы получите ссылку на WEB UI wg-easy а также 3 файлика в домашней дирректории хоста, переходим в нее командой ``` cd ``` 
и используем по необходимости через sh для ранее настроенного wg-easy сервиса в докер контейнере:

#### `sh run-wg`  - запустить (можно открыть и отредактировать другие необходимые параметры wg-easy)
#### `sh stop-wg` - остановить
#### `sh rerun-wg` - перезапусить (sh stop-wg + sh run-wg)


Установка: 
---
1. Купить wps сервер, подключиться по ssh

2. Скопировать и вставить в консоль следующую команду и следовать инстукциям.
```
curl -fsSL https://raw.githubusercontent.com/jinndi/wg-easy-bash/refs/heads/main/wg-easy-bash -o wg-easy-bash && bash wg-easy-bash
```
